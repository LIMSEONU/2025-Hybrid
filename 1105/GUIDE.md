### 📱 협업 도구 및 방법

#### 1. **Discord** (주요 협업 플랫폼)

**사용 목적**:
- ✓ 실시간 화면 공유 및 코드 리뷰
- ✓ 파일 공유 (HTML, CSS, JS, 이미지)
- ✓ 음성 채팅 (야간 작업 세션)
- ✓ 코드 스니펫 공유

**채널 구성**:
```
#general     - 일반 대화
#dev         - 개발 관련
#design      - 디자인 피드백
#bugs        - 버그 리포트
```

**선택 이유**: 
> "우리 셋 다 학교랑 과제 때문에 바빠서 직접 만나기 어려웠어요. 그래서 Discord로 실시간 소통하면서 작업했죠. 밤늦게까지 음성 채팅하면서 코딩하고 화면 공유로 바로바로 피드백 주고받았어요."

#### 2. **KakaoTalk** (빠른 소통)

**사용 목적**:
- ✓ 긴급 알림 및 빠른 의사결정
- ✓ 모바일 스크린샷 공유
- ✓ 회의 일정 조율
- ✓ 간단한 버그 제보

**선택 이유**:
> "급하게 확인해야 할 일이나 간단한 수정 사항은 카톡이 제일 빠르더라구요. '이거 색 좀 바꿔봐' 이런 식으로 바로바로 소통했어요."

#### 3. **GitHub** (버전 관리)

**사용 목적**:
- ✓ 코드 버전 관리
- ✓ GitHub Pages 배포
- ✓ 작업 이력 추적
- ✓ 코드 백업

#### 📅 Day 1: 프로젝트 킥오프 (2024.11.08)
**시간**: 저녁 8시 ~ 11시 (3시간)  
**장소**: Discord 음성 채팅

**논의 내용**:

1. **프로젝트 목표 설정**
   ```
   김종현: "야, 우리 박물관 사이트 하나 만들어볼까? 
           만 명한테 보여주는 거 목표로."
   
   권법진: "오 좋지. 근데 1주일 안에 가능해?"
   
   임선우: "AI 쓰면 될 듯. Claude랑 ChatGPT 활용하면 
           금방 만들 수 있을 거 같은데?"
   
   권법진: "ㅇㅇ 그럼 API 같은 거 말고 순수하게 
           HTML, CSS, JS만으로 해보자."
   ```

2. **역할 분담**
   ```
   김종현: "내가 기획이랑 회의록은 다 쓸게. 
           GitHub 관리도 내가 할게."
   
   권법진: "그럼 나는 메인 페이지랑 CSS, JS 전체를 담당할게. 
           BTS랑 오징어 게임 페이지도 내가 할게."
   
   임선우: "ㅇㅋ. 그럼 나는 K-Culture 페이지랑 
           README, PPT 만들게. 문서 정리도 내가 할게."
   
   권법진: "굿. 코드 퀄리티는 내가 책임질게."
   ```

3. **기술 스택 결정**
   ```
   권법진: "React 같은 거 쓸까?"
   
   김종현: "아니 너무 복잡해. 순수 JS가 나을 듯. 
            어차피 API도 안 쓰기로 했잖아."
   
   권법진: "ㅇㅋ 그럼 Vanilla JS로 가자. 
           대신 ES6+ 문법은 써도 되지?"
   
   임선우: "당연하지. 그리고 반응형은 필수야."
   ```

4. **개발 환경**
   ```
   임선우: "VS Code 쓰지?"
   
   권법진: "나 Cursor 써봤는데 진짜 좋더라. AI 자동완성 개쩔어."
   
   김종현: "오 그거 써보자. 그리고 나중에 Android Studio로 
            앱으로도 만들어볼까?"
   
   권법진: "Cordova로 하면 될 듯. 일단 웹부터 완성하자."
   ```

**결정 사항**:
- ✅ API 없이 순수 HTML/CSS/JS 사용
- ✅ AI 도구 적극 활용 (Claude, ChatGPT, Cursor)
- ✅ 1주일 내 완성 목표
- ✅ 중학생도 따라할 수 있게 문서화
